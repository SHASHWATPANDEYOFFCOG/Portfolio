## ✨ Overview

A handcrafted, dependency-free portfolio built with semantic **HTML5**, modern **CSS3**, and **vanilla JavaScript** — no frameworks and no build step. It's lightweight (~30&nbsp;KB), accessible, and fully responsive across all devices.

## 🚀 Features

- **Responsive design** — adapts cleanly from mobile to widescreen
- **Light / dark theme** toggle with smooth transitions
- **Smooth-scroll navigation** with active-section highlighting and a scroll progress bar
- **Scroll-reveal animations** and an animated typed-text hero
- **Animated stat counters** and skill progress bars
- **Client-side contact form** with live validation
- **SEO- & accessibility-friendly** — semantic markup, meta tags, and ARIA labels

## 🛠️ Tech Stack

| Layer    | Technology                                              |
| -------- | ------------------------------------------------------- |
| Markup   | HTML5 (semantic)                                        |
| Styling  | CSS3 — custom properties, flexbox, grid, animations     |
| Behavior | Vanilla JavaScript (ES6+)                               |
| Fonts    | Google Fonts — Poppins, JetBrains Mono                  |
| Hosting  | GitHub Pages                                            |

## 📂 Project Structure

```
Portfolio/
├── index.html      # Markup & content for every section
├── styles.css      # Theme, layout, and animations
├── script.js       # Interactivity: theme, scroll, typed text, counters, form
├── PORTFOLIO.jpg   # Profile image
└── .gitignore
```

## 💻 Run Locally

```bash
# Clone the repository
git clone https://github.com/SHASHWATPANDEYOFFCOG/Portfolio.git
cd Portfolio

# Then either open index.html directly in your browser,
# or serve it locally (recommended):
python -m http.server 8000
# Visit http://localhost:8000
```
